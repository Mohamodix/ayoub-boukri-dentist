# Ayoub Boukri Dental Clinic

A simple dentist appointment website.

## Features

- Home page
- Appointment booking
- Discord webhook notifications
- Vercel serverless API

## Deployment

1. Push to GitHub.
2. Import the repository into Vercel.
3. Add an environment variable:

Name:
DISCORD_WEBHOOK

Value:
Your Discord Webhook URL

4. Deploy.
